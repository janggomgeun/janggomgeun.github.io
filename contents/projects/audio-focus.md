# 개요
유튜브에서 음악을 듣다가 어느 순간에 다른 탭에서 동영상을 봐야하는 그 미묘한 순간이 귀찮아서, 현재 탭을 제외한 나머지 탭의 소리에 블러 효과를 넣어 버리는 크롬 확장 프로그램을 만들었었다.

# 사용된 기술
* 크롬 확장프로그램 개발환경 및 APIs
* Javascript, HTML, CSS
* Web Audio APIs
